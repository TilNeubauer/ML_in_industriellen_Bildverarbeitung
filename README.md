# Maschinelles Lernen in der industriellen Bildverarbeitung

Jede Uebung liegt getrennt unter `exercises/`. Abhaengigkeiten werden mit PDM
verwaltet. Nach der Installation startet Exercise 1.1 mit:

```powershell
pdm install
pdm run python exercises/01_unsupervised_learning/exercise_1_1_kmeans.py
```

Erzeugte Abbildungen werden unter `results/` abgelegt.
